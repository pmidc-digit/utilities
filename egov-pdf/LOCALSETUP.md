# Local Setup

To setup the egov-pdf service in your local system, clone the [utilities Service repository](https://github.com/egovernments/utilities).

## Dependencies

### Infra Dependency

- [ ] Postgres DB
- [ ] Redis
- [ ] Elasticsearch
- [ ] Kafka
  - [ ] Consumer
  - [ ] Producer

## Running Locally

```bash
- Provide host url in `HOST` property in config.js
- Run `npm install`
- Run `npm start`
```
